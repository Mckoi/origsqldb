The main (starting) class for a Mckoi database server.

Starting the Server

The database server is started by running java -jar mckoidb.jar

The META-INF/MANIFEST file in the jar file specifies the starting class as McKoiDBMain in this package, so java starts up the server application at the main method of McKoiDBMain.

The main method prints a banner, parses options, loads the config file, and eventually calls BootMain.boot to complete the startup. BootMain creates an instance of Database, initializes it, and finally starts the service that listens for and handles client connections.
